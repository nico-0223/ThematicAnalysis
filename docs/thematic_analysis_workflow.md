# Thematic analysis workflow

1. Initialise the database.
2. Load or edit a codebook.
3. Ingest raw conversations.
4. Segment turns conservatively.
5. Start an analysis run.
6. Complete or update Braun and Clarke phase records.
7. Apply rule-based annotations or import manual annotations.
8. Generate and review candidate themes.
9. Add memos, coding decisions, and adjudication records as needed.
10. Export reports.

The default demo follows this path through `make demo`.
